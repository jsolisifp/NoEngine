Funciones principales

bool NOE_Inicia()

Inicia el entorno NoEntiendo según la configuración presente en el fichero configuracion.ini.

Debe llamarse una única vez al inicio del programa.

Devuelve:

Cierto si el entorno se ha podido iniciar correctamente.

Falso en caso de error.

void NOE_Actualiza()

Actualiza el entorno NoEntiendo.

Debe llamarse periódicamente durante la ejecución del programa.

void NOE_Finaliza()

Finaliza el entorno NoEntiendo.

Debe llamarse una única vez al final del programa

void NOE_DibujaTilemap(int numTilemap)

Dibuja un tilemap en el buffer de pantalla .

Se dibujará en las coordenadas de mundo (0, 0).

Parámetros:

void NOE_DibujaSprite(int sprite, int x, int y, int ancho, int alto, bool invertirX, bool invertirY)

Dibuja un sprite en el buffer de pantalla

Parámetros:

sprite: Índice del sprite a dibujar.

x: Posición X donde se dibujará el sprite en coordenadas de mundo.

y: Posición Y donde se dibujará el sprite en coordenadas de mundo.

ancho: Ancho con que se dibujará el sprite en coordenadas de mundo o -1 para dibujarlo con el ancho original del sprite.

alto: Alto con que se dibujará el sprite en coordenadas de mundo o -1 para dibujarlo con el ancho original del sprite.

invertirX: 1 si queremos invertir horizontalmente el sprite y 0 si no.

invertirY: 1 si queremos invertir verticalmente el sprite y 0 si no.

void NOE_DibujaDecorado(int decorado)

Dibuja un decorado en el buffer de pantalla.

El decorado se dibujará en las coordenadas de pantalla (0 0) y se escalará hasta cubrirla por completo

Parámetros:

decorado: Índice del decorado a dibujar.

ancho: Ancho con que se dibujará el carácter en coordenadas de pantalla o -1 para dibujarlo con el ancho original del carácter en la fuente especificada.

alto: Alto con que se dibujará el carácter en coordenadas de pantalla o -1 para dibujarlo con el alto original del carácter en la fuente especificada.

fuente: Índice de la fuente con que se dibujará el carácter.

void NOE_RedimensionaDataset(int indiceDataset, int tamanyoMinimo)

Cambia el tamaño de un dataset.

Si el nuevo tamaño es mayor que el anterior, se rellenan con ceros los bytes añadidos, mientras que los anteriores mantienen su valor.

Si nuevo tamaño es menor, los bytes con offsets iguales o mayores que el nuevo tamaño pueden ser descartados y los bytes con offsets menores mantienen su valor.

Parámetros:

dataset: Índice del dataset que queremos redimensionar.

tamanyoMinimo: Nuevo tamaño en bytes que tendrá el dataset como mínimo.

int NOE_ObtenTamanyoDataset(int dataset)

Permite consultar el tamaño de un dataset.

Parámetros:

dataset: Índice del dataset cuyo tamaño queremos consultar.

Devuelve:

El tamaño del dataset en bytes.

void NOE_LimpiaDataset(int dataset, int desde, int hasta)

Pone un intervalo de bytes de un dataset a 0.

Parámetros:

dataset: Índice del dataset que queremos limpiar.

desde: Desplazamiento en bytes a partir del cual queremos limpiar.

hasta: Desplazamiento en bytes hasta el que queremos limpiar, incluído o -1 si se quiere limpiar hasta el final.

void NOE_CopiaDataset(int datasetOrigen, int origenDesde, int origenHasta, int destinoHasta)

Copia un intervalo de bytes de un dataset de origen a otro de destino, que puede ser el mismo.

Si el dataset de origen y destino es el mismo, los intervalos se pueden solapar.

Parámetros:

datasetOrigen: Índice del dataset que contiene los bytes a copiar.

datasetDestino: Índice del dataset en que se copiarán los bytes.

origenDesde: Desplazamiento en el dataset de origen del primer byte a copiar.

origenHasta: Desplazamiento en el dataset de origen del último byte a copiar.

destinoDesde: Desplazamiento en el dataset de destino en que se copiará el primer byte.

void NOE_RecuperaDatosDataset(int dataset, int desde, int hasta, void* bytes)

Recupera un intervalo de bytes almacenado en un dataset.

Parámetros:

dataset: Índice del dataset que almacena el intervalo de bytes.

desde: Desplazamiento en bytes dentro del dataset en que está el primer byte del intervalo.

hasta: Desplazamiento en bytes dentro del dataset en que está el último byte del intervalo.

bytes: Dirección a partir de la cual se dejarán los bytes recuperados.

void NOE_AlmacenaDatosDataset(int dataset, int desde, int hasta, const void* bytes)

Almacena un intervalo de bytes en un dataset, redimensionándolo si es necesario

Parámetros:

dataset: Índice del dataset en que se almacenará el intervalo de bytes.

desde: Desplazamiento en bytes dentro del dataset en que se almacenará el primer byte.

hasta: Desplazamiento en bytes dentro del dataset en que se almacenará el último byte.

bytes: Dirección en que están los bytes a almacenar.
